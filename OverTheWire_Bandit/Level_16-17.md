# Bandit Уровень 16 → 17

## Задача
Найти открытый порт в диапазоне 31000–32000, который принимает SSL/TLS-соединение, и получить SSH-ключ для входа на уровень 17.

## Что я сделал
1. Находился на уровне `bandit16`
2. Отсканировал порты: `nmap -p 31000-32000 localhost`
3. Обнаружил открытые порты: 31046, 31518, 31691, 31790, 31960
4. Проверил порт `31790` через `openssl s_client`, отправил текущий пароль
5. Получил RSA-ключ вместо пароля
6. Сохранил ключ в файл и использовал для подключения к `bandit17`:
   ```bash
   ssh -i sshkey17.private bandit17@bandit.labs.overthewire.org -p 2220
