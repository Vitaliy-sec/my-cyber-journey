# Bandit Уровень 14 → 15

## Задача
Отправить текущий пароль на порт 30000 локального сервера, чтобы получить пароль для следующего уровня.

## Что я сделал
1. Находился на уровне `bandit14`
2. Узнал текущий пароль: `cat /etc/bandit_pass/bandit14`
3. Отправил пароль на порт 30000 с помощью `netcat`:
   ```bash
   echo "MU4VWeTyJk8ROof1qqmcBPaLh7LDCPvS" | nc localhost 30000
Получил ответ с паролем для следующего уровня
